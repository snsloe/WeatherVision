# WeatherVision
# Weather Vision - Классификатор погоды по изображениям

## Описание проекта
Система автоматической классификации погодных условий (sunny, cloudy, rainy, foggy) на основе глубокого обучения. Достигнута точность более 90%.

## Архитектуры моделей
- **CNN** (кастомная архитектура)
- **MobileNetV2** (transfer learning)
- **EfficientNetB0** (transfer learning)

## Датасеты
1. **Multiclass Weather Dataset** (Kaggle) - 4 класса: sunrise, shine, rain, cloudy
2. **Дополнительный датасет** (Kaggle) - 4 класса: cloudy, rain, shine, sunrise

Объединенные классы:
- sunny (включает shine, sunrise)
- cloudy
- rainy
- foggy

## Результаты
- Лучшая модель: MobileNetV2
- Точность на валидации: >90%
- Время обучения: 15 эпох
- Размер изображений: 128×128

## Использование
1. **Обучение модели**: запустить notebook `project-weather-vision.ipynb`
2. **Запуск бота**: выполнить `python bot.py`
3. **Взаимодействие**: отправить фото в Telegram-бот для классификации

## Telegram-бот
- Определяет тип погоды на изображении
- Показывает уверенность предсказания
- Отображает топ-3 вероятных класса
